\# WatchHistoryExporter



A .NET Playwright console application to export personal TV Time watch history before the service shutdown.



\## Purpose



TV Time is shutting down, and this tool helps users export their personal watch history data before the service becomes unavailable.



\## Features



\- Browser automation using Playwright

\- Export Movies, TV shows and episodes

\- Preserve personal watch history

\- Save exported data locally



\## Requirements



\- .NET 8 SDK

\- A TV Time account



\## Usage



1\. Clone the repository

2\. Run the application

3\. Enter your TV Time credentials when prompted

4\. The exporter will collect your data and save the results locally



\## Disclaimer



This project is intended for users exporting their own personal data. It is not affiliated with TV Time.
